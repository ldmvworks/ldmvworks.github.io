Works

制作領域What We Build

ゲーム本体だけでなく、プレイ体験を支えるUI、演出、公開用素材までを同じ方向性で設計します。 The studio designs the game, UI, presentation, and public-facing assets as one connected experience.

01

ゲーム企画Game Planning

戦術判断、部隊編成、報酬選択など、短いサイクルで意思決定が続くゲーム設計。 Game systems built around tactical choices, unit setup, rewards, and compact decision loops.

02

PCゲーム開発PC Game Development

PC環境での操作性、視認性、デモ版公開を前提にしたゲーム実装。 Implementation for PC play, readable screens, and demo-ready development milestones.

03

ゲームUI / UXGame UI / UX

カード、ステータス、編成、戦況を読みやすく伝える画面設計とUI素材制作。 UI and screen design for cards, stats, formation, and battle state readability.

04

効果音・デジタル素材Sound & Assets

操作感や状況変化を補強する効果音、演出、ストアページ向け素材の制作。 Sound effects, presentation assets, and store-page materials that support the game experience.

Project

開発中タイトルProject in Development

現在、戦術判断・部隊運用・ローグライト要素を含むPC向けゲームを開発中です。 2026年10月のSteam Next Festに向けて、20〜30分程度で基本サイクルを体験できるデモ版を制作しています。 A PC game with tactical decisions, unit management, and roguelite elements is currently in development. The current goal is a 20-30 minute demo for Steam Next Fest in October 2026.

Process

制作プロセスProduction Process

  1. 01

    Prototype

    プレイの核になる判断、報酬、編成サイクルを試作します。 Prototype the core decisions, rewards, and formation loop.

  2. 02

    Play Loop

    20〜30分で基本サイクルを体験できるデモ版へ整理します。 Shape the game into a demo with a 20-30 minute core loop.

  3. 03

    Presentation

    タイトル画面、UI、効果音、公開用素材を磨き込みます。 Refine the title screen, UI, sound, and public-facing assets.
