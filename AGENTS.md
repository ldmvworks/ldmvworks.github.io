# Deployment policy

- Publish this repository and all demos only through GitHub Pages at `https://ldmvworks.github.io/`.
- Never create, save, deploy, or update an OpenAI Sites / `chatgpt.site` deployment for this repository.
- Do not add `.openai/hosting.json` or re-enable the Sites Vite plugin.
- Push demo updates to the GitHub `origin` remote. The `main` branch is the GitHub Pages source.
- Sites management tools may be used only to disable or remove the legacy deployment, unless the user explicitly reverses this policy.
